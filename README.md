# OS-Development-from-scratch

**System Requirement**
Linux OS
GCC - GNU C compiler
NASM - Assembly
QEmu - Emulator
GDB - Debugging tool

**How to Run**
1. Go to specific folder from list
2. run command to build: make
3. command to run OS in Emulator: qemu-system-i386 -fda os-image.bin

**Debug**
command: make debug
